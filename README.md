# NodeVideoStream

1- descargar el proyecto.  
2- instalar nodejs   
3- ir a directorio del proyecto abrir una terminal y ejecutar el comando:   
```bash
npm install 
```
4- ejecutar el comando:  
```bash
node app.js  
```
5- abrir en http://localhost:3000/index.html  
6- emitir video llamada.
